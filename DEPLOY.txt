ИНСТРУКЦИЯ ПО ДЕПЛОЮ НА RAILWAY.COM

ВАЖНО: Теперь запускается ТОЛЬКО web.py, который автоматически запустит бота!

1. Загрузите на GitHub:
   git add .
   git commit -m "Bot with web panel"
   git push origin main

2. Зайдите на https://railway.app
   - Войдите через GitHub
   - New Project → Deploy from GitHub repo
   - Выберите ваш репозиторий

3. Railway автоматически:
   - Установит зависимости из requirements.txt
   - Запустит web.py (который запустит бота автоматически)
   - Даст вам публичный URL

4. Получите URL панели:
   - Railway покажет URL: https://ваш-проект.up.railway.app
   - Этот URL доступен с ЛЮБОГО устройства в мире!

5. Откройте панель:
   - Вставьте URL в браузер (с компьютера, телефона, планшета)
   - Введите пароль: pon4ik6662
   - Готово!

ВАЖНО:
- Бот работает 24/7
- Панель доступна отовсюду
- Защищена паролем
- JSON файлы сбросятся при обновлении (используйте Railway Volumes)

ФАЙЛЫ:
- web.py - главный файл (запускает всё)
- bot.py - Discord бот (запускается автоматически)
- templates/ - HTML страницы
- static/js/ - JavaScript
- requirements.txt - зависимости
- Procfile - команда запуска (web: python web.py)
